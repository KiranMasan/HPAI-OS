# 🚀 HPAI-OS (Human Progress AI Operating System)

## AI-Powered Personal Growth & Learning Platform

HPAI-OS is an AI-powered Human Development Operating System designed to help users achieve their learning, productivity, and career goals through personalized AI guidance.

Unlike traditional AI assistants that only answer questions, HPAI-OS creates personalized growth roadmaps, adaptive daily missions, digital twin profiles, progress analytics, and voice-based interactions.

---

## 🎯 Problem Statement

Many students and professionals struggle with:

* Lack of structured learning plans
* Poor consistency and productivity
* Difficulty tracking progress
* Lack of personalized career guidance

HPAI-OS solves these challenges using Artificial Intelligence by automatically generating roadmaps, tasks, recommendations, and progress analytics.

## 🚀 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Ollama

```bash
ollama serve
ollama pull mistral
```

### Frontend

```bash
cd frontend
flutter pub get
flutter run
```
